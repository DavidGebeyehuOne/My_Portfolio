   ```bash
   git clone https://github.com/DavidGebeyehuOne/My_Portfolio.git
   ```

2. **Enter the Project Directory**
   ```bash
   cd My_Portfolio
   ```

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Start the Development Server**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:3000`.
